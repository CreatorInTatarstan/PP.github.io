Система контроля версий

Необходима для совместной работы над проектом с командой, возможность делиться кодом и вкладами, и избегать редакционных конфликтов. Сейчас Git является наиболее популярным инструментом контроля версий, и репозиторий кода Github, основанный на Git, также является очень популярным.

Система автоматизации

Система автоматизации, такая как Gulp, для автоматического выполнения повторяющихся задач, например, минимизации кода, запуска тестов и автоматическое обновление браузера.
